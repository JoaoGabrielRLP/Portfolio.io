Sobre Mim


Atualmente, atuo como Analista de Suporte em Tecnologia, onde desenvolvi habilidades essenciais como resolução de problemas, pensamento crítico e criatividade. Essa experiência tem sido fundamental para minha transição para a área de desenvolvimento de software, permitindo que eu compreenda melhor as necessidades dos usuários e proponha soluções eficientes.


Participei de projetos nos quais identifiquei problemas e desenvolvi ou documentei soluções, visando otimizar e facilitar o atendimento no suporte.


Tenho experiência em desenvolvimento web e back-end, com projetos disponíveis no meu GitHub. No desenvolvimento web, utilizo HTML, CSS e JavaScript, além de frameworks como Bootstrap e Vue.js para criar interfaces modernas e responsivas. No back-end, trabalho com PHP, Java, C e bancos de dados SQL, garantindo a estrutura e a lógica por trás das aplicações.


Meu objetivo é crescer na área de desenvolvimento de software e aplicativos, aplicando minhas habilidades para contribuir com soluções inovadoras e impactantes.
